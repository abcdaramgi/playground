티스토리 첫화면에 사이드바 숨기기

by 영감은어디에 2023. 1. 16.
티스토리
body.id 값이 홈화면일때 실행 
 
if (document.body.id == "tt-body-index"){
   $("#aside").css("display", "none");
} else {
   $("#aside").css("display", "block");
}
 
 
 
코드치환자 
페이지 타입 body_id
홈화면 tt-body-index
글화면 tt-body-page
카테고리 글 리스트 tt-body-category
보관함 글 리스트 tt-body-archive
태그 리스트 tt-body-tag
검색결과 리스트 tt-body-search
방명록 tt-body-guestbook
지역로그 tt-body-location
